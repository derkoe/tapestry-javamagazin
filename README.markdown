# Tapestry Javamagazin

This project contains the examples to the article series in German magazine [Javamagazin](www.java-magazin.de).

The demo applications are deployed to [Cloud Foundry](http://cloudfoundry.com/):
 * http://tapestry-javamagazin1.cloudfoundry.com/
   This demo shows basic component and event usage.
 * http://tapestry-javamagazin2.cloudfoundry.com/
   A simple CRUD demo using Tapestry's Grid and BeanEditor components.

## Developer Info

This project is built with Maven - to generate the jar run: mvn package 

Continuous builds are running on [Travis](http://travis-ci.org/): [![Build Status](https://secure.travis-ci.org/derkoe/tapestry-javamagazin.png?branch=master)](http://travis-ci.org/derkoe/tapestry-javamagazin)
